Description

Welcome to the RAND co.op! This is a simple web application that allows you to evaluate boolean equations. You can enter a boolean equation in the input field and the calculator will evaluate the equation and display the result. The calculator supports the following operators: AND, OR, NOT, NAND and XOR. You can also use parentheses to group expressions and change the order of operations. Try it out and see how it works!

Instructions

To use the calculator, enter a boolean equation in the input field and press the "Calculate" button. The calculator will evaluate the equation and display the result.
